About the project

This is my final project submission for Database Management Systems at the University of Arkansas. You can read more about the project on the About page or view the source code on Github.


The assignment

  1. Add a new student to the Student table
  2. Add a course to the Course table
  3. Add an application to the Enrollment table
    • Click "View" on a student in the student list
    • Click enroll
    • Select the course(s) to enroll the student in and click submit.
  4. View all students
  5. View all courses for a given department
    • Click "View" on a department in the department list
  6. View all courses for a given student
    • Click "View" on a student in the student list
